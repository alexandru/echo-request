Echo HTTP

Web service publicly available for mirroring HTTP requests:

1. Show My Request Headers

GET https://echo.i64.app/all
This also accepts an "ip" parameter, for overriding your IP: 
GET https://echo.i64.app/all?ip=185.210.218.104
Test via cURL: 
curl 'https://echo.i64.app/all' -H 'Host: echo.i64.app' -H 'User-Agent: Mozilla/5.0 (Macintosh; Intel Mac OS X 10.13; rv:60.0) Gecko/20100101 Firefox/60.0' -H 'Accept: text/html,application/xhtml+xml,application/xml;q=0.9,*/*;q=0.8' -H 'Accept-Language: en-US,en;q=0.5' --compressed -H 'Connection: keep-alive' -H 'Upgrade-Insecure-Requests: 1' -H 'Cache-Control: max-age=0'

2. Show My IP

GET https://echo.i64.app/ip
Test via cURL: 
curl 'https://echo.i64.app/ip' -H 'Host: echo.i64.app'

3. Geo IP

GET https://echo.i64.app/geoip
This also accepts an "ip" parameter, for overriding your IP: 
GET https://echo.i64.app/geoip?ip=185.210.218.104
Test via cURL: 
curl 'https://echo.i64.app/geoip' -H 'Host: echo.i64.app'
