Echo HTTP

Web service publicly available for mirroring HTTP requests:

1. Show My Request Headers

GET https://echo.fp4.dev/all
This also accepts an "ip" parameter, for overriding your IP: 
GET https://echo.fp4.dev/all?ip=185.210.218.104

2. Show My IP

GET https://echo.fp4.dev/ip

3. Geo IP

GET https://echo.fp4.dev/geoip
This also accepts an "ip" parameter, for overriding your IP: 
GET https://echo.fp4.dev/geoip?ip=185.210.218.104

4. Timeout

Endpoint for simulating a timeout. Example 1: 
GET https://echo.fp4.dev/timeout/3seconds
Example 2: 
GET https://echo.fp4.dev/timeout/1minute
The syntax for the given parameter is `<number><unit>`.
